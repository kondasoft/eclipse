# Eclipse

Eclipse is a modern, performance-first Shopify theme designed for speed, flexibility, and a clean user experience. Built from scratch, it avoids heavy frameworks and unnecessary code, resulting in a fast, efficient storefront.

The theme includes a refined Light and Dark mode system, allowing customers to manually switch between modes or automatically sync with their device preferences for a seamless browsing experience.

This free edition provides a strong core foundation for modern Shopify stores. Premium and advanced features are available in our paid themes for merchants who need more advanced functionality and customization.

## Highlighted features

- **Light & Dark Color Schemes** – Built-in color switcher with automatic device detection
- **Modern & Performance-Focused** – Carefully crafted from scratch without heavy frameworks
- **Extremely Fast** – Optimized codebase with no boilerplate or unnecessary scripts
- **Accessibility Ready** – Designed with accessibility standards and screen-reader support
